Codejock Software Release Notes

Codejock Software's release notes include updated information for the documentation provided with the product line. The information contained in this document may be more up-to-date than the information that is provided in the help system.

Notice on ActiveX fixes: In order to eliminate duplicated records the fixes are not repeated in SuitePro/ActiveX blocks because all ActiveX components are built on top of ToolkitPro components, which means every internal ToolkitPro or component specific fix is applicable for ActiveX components as well. Only ActiveX specific fixes/changes are listed separately in the change log.
